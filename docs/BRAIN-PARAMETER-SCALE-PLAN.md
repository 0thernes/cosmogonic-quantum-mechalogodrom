## Honesty contract

- **Live JS runtime** caps organ allocation at `LIVE_NODE_CAP` (4096) — the browser runs the full
  deterministic math today; designed scale is architecture accounting.
- **5M parameters** = near-term GPU/native roadmap (DataTexture / C++ backend).
- **1 billion neurons** = ultimate `SCALE_MASSIVE` target — declared, tested, not live-allocated in TS.
- **100 thought variations** = psychology · neuroscience · quantum · neuromorphic · wet · A-Life ·
  empowerment substrates (`APEX_THOUGHT_VARIATIONS`) — metadata for 1/1 alien cognition, not Level-5
  consciousness claims.
