## §A. Frame data-flow — how one tick happens

`World.step(dt)` (the composition root, [`src/world.ts`](../src/world.ts)) runs the pipeline each frame.
Heavy work is **cadenced** (runs on slow frames that scale with population) so the budget holds at scale:

```
input → audio analysis → weather → puppeteers → grid rebuild (every 2f)
  → shoggoths → titans → leviathans → super-creature body → singularities
  → entities.update (births/deaths/behaviour/motion)
  → NHI minds (every 18f) · super-creature mind (every 4f) · connectome (cadence by population)
  → quantum circuit (30f) · economy clear (30f, after war→sanctions) · reaction-diffusion
  → graph communities (240f) / rank (600f) · atmosphere · environment · artifacts
  → telemetry panel (8f) · Observatory + Market + ARCHITECT + Superhero HUD (18f)
  → instanced pool sync (LAST, after every visual mutation) → render
```

Every randomness draw flows through a seeded `Rng` ([`src/math/rng.ts`](../src/math/rng.ts)); the economy
and super-creature use **isolated sub-streams** so they never perturb the core golden. See
[ARCHITECTURE-2026-06-26.md](./ARCHITECTURE-2026-06-26.md) for the full pipeline.

## §B. Build, run & verify

- **Dev**: `bun dev` (cwd MUST be the repo root — the Bun HTML-import bundler fails elsewhere). App at
  `:3000`, diagrams at `/docs`, spec at `/spec`, lab at `/lab`.
- **Gate** (before every commit): `bun run check` — prettier → `tsc` strict → oxlint → `bun test` → build.
- **Bench**: `bun run bench`; record new hot paths in [BENCHMARKS-2026-06-26.md](./BENCHMARKS-2026-06-26.md).
- **File map**: `bun scripts/gen-filemap.ts` regenerates [FILE-MAP.md](./FILE-MAP.md).
- **Native**: `pwsh native/build.ps1 -Shot` (MinGW + CMake; renders 4K plates).

## §C. Troubleshooting

| Symptom                               | Cause → fix                                                                                                                                                                                                   |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Blank app / nothing renders           | Bundler run from the wrong cwd — run `bun dev` from the repo root.                                                                                                                                            |
| Determinism test fails after a change | A new `Math.random`/`Date.now` in sim logic, or a stray `Rng` draw shifting the order — route randomness through `src/math/rng.ts` and keep new systems on a sub-stream.                                      |
| The ✦ AI is silent                    | It's offline/rate-limited — open its 🩺 diagnostics for the per-provider reason + restart; or use ❓ HELP (offline, repo-grounded). See [COPILOT-PROVIDERS-2026-06-26.md](./COPILOT-PROVIDERS-2026-06-26.md). |
| Native build segfaults                | See ADR-0007 — Jolt callbacks + member-init order; build with MinGW UCRT.                                                                                                                                     |
| Path errors on Windows                | This repo's path contains `[ ]` — use `-LiteralPath` in PowerShell, or the dedicated file tools.                                                                                                              |
