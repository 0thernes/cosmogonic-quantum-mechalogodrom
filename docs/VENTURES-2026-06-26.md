# Ventures — product horizon (aspirational)

This document describes **future product surfaces** beyond the current single-player browser
instrument. The live simulation (v0.x) remains **local, deterministic, and PII-free** per
[ROADMAP-2026-06-26.md](../ROADMAP-2026-06-26.md) non-goals for the core sim. Everything below is
**Horizon: Ventures** — planned layers that wrap the dome without compromising sim determinism.

## Scope split

| Layer | Today (v0.x) | Ventures horizon |
| ----- | ------------ | ---------------- |
| Simulation | Single machine, seeded Rng, no accounts | Same core — still deterministic |
| Identity | None | Optional email / OAuth for saves & drops |
| Multiplayer | Non-goal | Up to 10 concurrent players (authoritative server) |
| Payments | In-sim wallet only | Stripe donations + $10 early / $5 EDU tier |
| Outreach | NHSI dashboard receipts | Institution CRM + cold-contact list |

## Milestones (ordered)

1. **Hosted deploy** — GitHub Pages + optional VPS for API (`/api/waitlist` scaffold live).
2. **Waitlist & contact** — email capture, EDU verification, GitHub-star perks (manual fulfillment).
3. **Persistent profiles** — cloud save of non-deterministic meta (settings, unlocks) — never replay seed.
4. **Multiplayer dome** — 10-player rooms; sim state authoritative on server; clients interpolate.
5. **Monetization** — donations, early-access subscription, in-game cosmetic purchases.
6. **Peer-review meta-analysis** — falsifiable study packet (repo + sim + dome metrics).
7. **Institution outreach** — curated list: labs, professors, foundations (name, title, email, focus).
