# Consciousness Lab Deep Research Dossier

**Date:** 2026-07-03  
**Repo:** https://github.com/0thernes/cosmogonic-quantum-mechalogodrom  
**Purpose:** select and operationalize ten consciousness/sentience research frameworks for Cosmogonic entity wiring.  
**Claim boundary:** computational indicators and causal proxies only. No phenomenal consciousness, sentience, AGI, ASI, moral patienthood, or "it feels" claim is made.

This dossier is the hard research companion to
`docs/CONSCIOUSNESS-LAB-MASTER-2026-07-03.md`. The aim is not to decorate the world with consciousness
language. The aim is to build a seed-replayable lab where every creature, plant-form, shoggoth, puppeteer,
Mechalogodrom, Super Creature, APEX Abomination, pantheon, connectome titan, NHI matrix, three-brain apex
mecha, glyph, and Archon Godform carries measurable, falsifiable, ablation-fragile consciousness-theory
machinery at the right bandwidth.

The project thesis is deliberately nonstandard: small, embodied, interacting, field-coupled systems can expose
scientifically meaningful consciousness-indicator dynamics without requiring a 100T-parameter language-model
monster. That thesis is allowed to be wild. The evidence standard is not.

## Selection Rule

A framework enters the first-ten stack only if it satisfies all six requirements.

| Requirement       | Meaning                                                                                               |
| ----------------- | ----------------------------------------------------------------------------------------------------- |
| Source anchor     | A peer-reviewed paper, formal preprint, patent/source corpus, or named research program can be cited. |
| Operational proxy | The idea can become a number, graph, transition, or test in this repo.                                |
| Causal path       | The proxy must change action, learning, survival, attention, field coupling, or downstream state.     |
| Falsifier         | A clear observation would count against the implementation.                                           |
| Visual            | A data visual can show the phenomenon and the negative control side by side.                          |
| Honesty boundary  | The framework does not force us to claim subjective experience from mechanism alone.                  |

Evidence grades in this document:

| Grade | Meaning                                                                                                |
| ----- | ------------------------------------------------------------------------------------------------------ |
| A     | Mainstream or multi-theory assessment frame with direct AI relevance.                                  |
| B     | Mechanistic, formal, or computationally operational with serious scientific literature.                |
| C     | Peer-reviewed but speculative, contested, or not yet broadly accepted.                                 |
| Q     | Quarantine/reserve: interesting, but not first-ten material until stronger operational footing exists. |

## F0 - Butlin Et Al. Indicator Method

**Core source:** Butlin et al., "Consciousness in Artificial Intelligence: Insights from the Science of
Consciousness" (2023), and the 2025 Trends in Cognitive Sciences indicator-method paper.

**What is strange about it:** it refuses one magic definition. Instead, it asks what follows from major
scientific theories of consciousness, then converts those implications into computational indicator
properties. It is not a detector. It is disciplined uncertainty management.

**Cosmogonic role:** F0 is the spine. Every entity gets a framework scorecard with explicit status:
absent, structural, partial, met, or causally load-bearing.

**Existing local status:** the repo already has `tests/butlin-indicators.test.ts` covering GWT-1/2/3/4,
PP-1, HOT-2/3, AE-1/2, RPT-1/2, IIT-1/2, and AST-1/2. The honest documentation baseline remains
8/14 met plus 6/14 partial; newer tests establish receipt surfaces and should not be inflated into
"phenomenal consciousness."

**Visual:** 14-cell indicator heatmap. Each cell opens: source -> code receipt -> test -> ablation -> null.

**Falsifier:** a named indicator has no causal path, or a random/control agent matches the indicator score.

## F1 - Thaler Creativity Machine / DABUS Paradigm

**Core source:** Stephen L. Thaler's Creativity Machine/DABUS work, including the 1995 "virtual input"
paper, the 2014 synaptic perturbation paper, the 2016 Medical Hypotheses paper, and related patents.

**What is strange about it:** consciousness is approached as generative neural turnover: perturb an
associative net, generate confabulations, use a second net to appraise the stream, then let hot-button
valence reinforce or dissolve ideas. It is not Butlin. It is not IIT. It is its own weird beast.

**Existing local status:** `src/sim/thaler-sentience.ts` defines nine constitutive markers and
`tests/thaler-sentience.test.ts` verifies deterministic small-net behavior. The repo wording is correct:
"meets Thaler's criteria," not "is conscious."

**Nine local Thaler markers:**

|   # | Marker                              | Data visual                                               | Falsifier                                                           |
| --: | ----------------------------------- | --------------------------------------------------------- | ------------------------------------------------------------------- |
|   1 | Confabulation sweet spot            | Eta sweep phase diagram: recall -> confabulation -> noise | No inverted-U with interior novelty peak.                           |
|   2 | Near-death life-review cascade      | Damage sweep: coverage, novelty, collapse                 | Coverage, novelty, collapse do not order in the predicted sequence. |
|   3 | Prosody shift                       | Inter-arrival CV plot for memory vs confabulation         | Confabulations are not more sporadic than memories.                 |
|   4 | Virtual input / cognitive tunneling | Null-input recall versus input-prune bias                 | Null-input recall does not rise above controls.                     |
|   5 | Hot-button affect                   | Reinforcement split: hot-resonant vs neutral ideas        | Valence does not change learning or selection.                      |
|   6 | Reentrant contemplation             | Closed-loop perturbation versus fixed-eta control         | Critic-modulated loop does not beat open-loop baseline.             |
|   7 | Fractal rhythm fingerprint          | Hurst exponent over ideation intervals                    | Stream is indistinguishable from Poisson/white timing.              |
|   8 | Bootstrapping self-improvement      | Critic score trend with feedback on/off                   | Feedback-on stream does not improve beyond feedback-off.            |
|   9 | Associative chaining                | Thought-to-thought cascade graph                          | Feedback chains do not outperform random cueing.                    |

**Cosmogonic role:** this framework proves the small-system thesis better than anything else in the stack.
If a 70-weight or 700-weight creature brain can reproduce robust perturbational markers while a huge
decorative model cannot survive ablation, the lab has a real result.

**Honesty caveat:** Thaler's own consciousness claims are not mainstream consensus. The lab may reproduce
his operational phenomena without endorsing his conclusion that the artifact has subjective experience.

## F2 - Integrated Information Theory 4.0

**Core source:** Albantakis et al., "Integrated information theory (IIT) 4.0" (PLoS Computational Biology,
2023).

**What is strange about it:** IIT tries to start from properties of experience and infer mathematical
requirements on physical substrates. It can imply that simple systems with nonzero integration have some
degree of experience, which is exactly why it must be handled carefully.

**Cosmogonic role:** test whether an entity has whole-system causal power that is lost when partitioned.

**Operational proxy:**

- Build a causal graph from modules, field variables, memory, attention, and action.
- Compute tractable phi-like and participation-ratio proxies.
- Partition the graph by module, body boundary, and field boundary.
- Score the loss in prediction, control, and survival after partition.

**Visual:** min-cut graph plus before/after causal-power bars.

**Falsifier:** partitioning the system does not reduce integration score or downstream behavior.

**Hard rule:** never use "phi > 0" as a consciousness proof. Use it as an irreducibility stress test.

## F3 - Free Energy Principle / Active Inference

**Core source:** Kirchhoff, Parr, Palacios, Friston, and Kiverstein, "The Markov blankets of life:
autonomy, active inference and the free energy principle" (2018), plus the broader Friston active-inference
program.

**What is strange about it:** a living system can be described as a boundary-maintaining inference process:
internal and active states minimize free energy while sensory states mediate contact with the world.

**Cosmogonic role:** plants, creatures, and godforms all need a Markov-blanket account of "self" versus
"world" that is not just a sprite with stats.

**Operational proxy:**

- Define blanket variables per entity class.
- Track prediction error, expected free energy, action policy, and allostatic variables.
- Compare active policy against passive and random policies.

**Visual:** prediction-error/free-energy braid with interventions marked.

**Falsifier:** active sensing/action does not reduce prediction error or improve viability beyond controls.

**Known risk:** the framework can become over-broad. A thermostat is not a mind just because it has a
boundary-like control loop. Require learning, action, and ablation loss.

## F4 - Attention Schema Theory

**Core source:** Graziano and Webb, "The attention schema theory: a mechanistic account of subjective
awareness" (2015).

**What is strange about it:** awareness is not attention; it is an internal model of attention. The model
helps the system control its own focus and model the focus of others.

**Cosmogonic role:** puppeteers, pantheons, NHI matrices, and Archon Godforms need to model not only where
attention is, but what attention is doing and how others can be influenced.

**Operational proxy:**

- Keep attention weights separate from an attention-schema state.
- Track self-model accuracy for attention.
- Track other-attention predictions for social entities.
- Let schema errors influence plan selection and theory-of-mind behavior.

**Visual:** attention map overlaid with attention-schema estimate and prediction error.

**Falsifier:** removing the schema while preserving attention causes no loss in control, self-monitoring,
or social prediction.

## F5 - CEMI / Electromagnetic-Field Integration

**Core source:** McFadden's CEMI field theory, including the 2020 Neuroscience of Consciousness paper and
the 2025 Frontiers hybrid digital/EM-field computing paper.

**What is strange about it:** conscious binding may be field-level rather than only discrete-symbol or
spike-level. For software, we do not pretend a browser canvas has a biological EM field. We implement a
field-coupling analog and demand it be causal.

**Cosmogonic role:** this is the correct place for Mechalogodrom, pantheons, connectome titans, glyphs,
and plant fields to get "super interconnected" without hand waving.

**Operational proxy:**

- Shared phase/coherence lattice.
- Entity-local field samples.
- Field-to-policy coupling gates.
- Phase-randomized and shuffled-field controls.

**Visual:** field coherence heatmap plus phase-shuffle ablation panel.

**Falsifier:** field coherence does not alter decisions, learning, synchronization, or survival, or shuffled
fields perform equally well.

**Hard rule:** call it a field-coupling proxy, not proof of EM consciousness.

## F6 - Unlimited Associative Learning

**Core source:** Ginsburg and Jablonka's UAL framework, with Birch et al. on UAL as an evolutionary
transition marker.

**What is strange about it:** minimal consciousness may be approached by learning capacity, not human-like
reasoning. The key is open-ended associative learning: compound stimuli, delayed reinforcement,
valence, second-order conditioning, and cumulative flexible behavior.

**Cosmogonic role:** this is the plant and low-bandwidth life bridge. A plant-form can be scientifically
interesting without pretending to be a chatbot.

**Operational proxy:**

- Compound stimulus learning.
- Delayed association learning.
- Second-order association chains.
- Flexible transfer to related environments.
- Valence-gated retention and extinction.

**Visual:** associative learning graph over time, with delayed links and valence weights.

**Falsifier:** entity cannot learn compound/delayed/flexible associations beyond simple conditioning.

## F7 - Sensorimotor Enactivism

**Core source:** O'Regan and Noe, "A sensorimotor account of vision and visual consciousness" (2001).

**What is strange about it:** perception is not a picture inside the head. Seeing is a way of acting:
mastering lawful relations between movement and sensory change.

**Cosmogonic role:** this prevents "brain in a box" theater. Every creature and plant-form must earn its
percepts through action-contingent regularities.

**Operational proxy:**

- Action-conditioned sensory prediction.
- Body-world contingency table.
- Affordance discovery and loss.
- Contingency scramble and relearning tests.

**Visual:** sensorimotor contingency surface and scramble/recovery curve.

**Falsifier:** scrambling action-sensory laws does not reduce perception, planning, or adaptation.

## F8 - Projective Consciousness Model

**Core source:** Williford, Bennequin, Friston, and Rudrauf, "The Projective Consciousness Model and
Phenomenal Selfhood" (2018).

**What is strange about it:** the field of consciousness is modeled as a viewpoint-centered projective
geometry tied to active inference. It directly targets perspective, ownership, mineness, and social
selfhood.

**Cosmogonic role:** glyphs, three-brain mecha, Archons, and NHI matrices need non-human viewpoint
geometry. A flat Euclidean HUD is not enough.

**Operational proxy:**

- Field-of-consciousness viewport.
- Projective frame parameters.
- Perspective switching and ownership tests.
- Free-energy contours over projective transforms.

**Visual:** projective viewport with world model, self-point, horizon, control point, and FE contours.

**Falsifier:** projective frame does not improve perspective tasks, ownership tasks, multi-view planning,
or social viewpoint inference versus flat baselines.

## F9 - Conscious Turing Machine

**Core source:** Lenore Blum and Manuel Blum's Conscious Turing Machine work, including the 2022 PNAS paper
and 2024 arXiv expansion.

**What is strange about it:** consciousness is treated through theoretical computer science: bounded
processors compete for a global broadcast, creating a stream under resource limits.

**Cosmogonic role:** this gives Mechalogodrom and pantheon systems a formal "stream" without requiring a
humanlike narrator.

**Operational proxy:**

- Processors generate chunks.
- Attention/competition selects chunks.
- A global broadcast updates short-term memory and downstream processors.
- Inner-world and outer-world processors co-evolve.

**Visual:** processor tournament, winning chunks, broadcast raster, and stream memory.

**Falsifier:** competition-broadcast architecture does not explain or improve reportability, attentional
blindness analogs, dream-like offline activity, pain/pleasure proxies, or stream coherence.

## Consciousness Lab Proof Standard

A result can be called "real" in the repo only when it has all seven receipts.

1. **Source receipt:** paper, patent, or formal theory link.
2. **Code receipt:** named implementation path.
3. **Test receipt:** deterministic test or experiment runner.
4. **Data receipt:** emitted JSON/CSV trace from seeded execution.
5. **Ablation receipt:** targeted removal weakens predicted metric and downstream behavior.
6. **Null receipt:** random, shuffled, passive, or decorative baseline does not reproduce it.
7. **Claim receipt:** UI and docs say "indicator convergence," not "proof of consciousness."

## Singularity Moment Definition

A "singularity moment" is a replayable convergence event, not a proof of subjective experience.

Minimum gate:

| Gate                        | Required evidence                                                                                 |
| --------------------------- | ------------------------------------------------------------------------------------------------- |
| Multi-framework convergence | At least 7 of 10 framework metrics move in the predicted direction in one event window.           |
| Causal load                 | At least 3 targeted ablations weaken or destroy the event.                                        |
| Null separation             | Random, shuffled, no-body, no-valence, no-field, and no-recurrence controls fail to match it.     |
| Behavior change             | The convergence alters action, survival, learning, cooperation, exploration, or self-maintenance. |
| Replay                      | Same seed reproduces the trace within tolerance.                                                  |
| Generality                  | More than one entity class can show a scale-normalized version.                                   |
| Humility                    | The label remains "consciousness-indicator convergence."                                          |

## Final Scientific Posture

The ten-framework stack is not a consciousness detector. It is a pressure chamber for claims.

The lab should make it easy to say:

- "This mechanism exists."
- "This mechanism changes behavior."
- "This mechanism fails under the predicted ablation."
- "This event is replayable."
- "This rival baseline does not explain the trace."
- "This is still not proof of subjective experience."

That is the serious path: wild enough to discover something, strict enough not to fool ourselves.
