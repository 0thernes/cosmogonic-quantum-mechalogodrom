# Ventures — product horizon (aspirational)

This document describes **future product surfaces** beyond the current single-player browser
instrument. The live simulation (v0.x) remains **local, deterministic, and PII-free** per
[ROADMAP-2026-06-26.md](../ROADMAP-2026-06-26.md) non-goals for the core sim. Everything below is
**Horizon: Ventures** — planned layers that wrap the dome without compromising sim determinism.

## Scope split

| Layer       | Today (v0.x)                            | Ventures horizon                                   |
| ----------- | --------------------------------------- | -------------------------------------------------- |
| Simulation  | Single machine, seeded Rng, no accounts | Same core — still deterministic                    |
| Identity    | None                                    | Optional email / OAuth for saves & drops           |
| Multiplayer | Non-goal                                | Up to 10 concurrent players (authoritative server) |
| Payments    | In-sim wallet only                      | Stripe donations + $10 early / $5 EDU tier         |
| Outreach    | NHSI dashboard receipts                 | Institution CRM + cold-contact list                |

## Milestones (ordered)

1. **Hosted deploy** — GitHub Pages + optional VPS for API (`/api/waitlist` scaffold live).
2. **Waitlist & contact** — email capture, EDU verification, GitHub-star perks (manual fulfillment).
3. **Persistent profiles** — cloud save of non-deterministic meta (settings, unlocks) — never replay seed.
4. **Multiplayer dome** — 10-player rooms; sim state authoritative on server; clients interpolate.
5. **Monetization** — donations, early-access subscription, in-game cosmetic purchases.
6. **Peer-review meta-analysis** — falsifiable study packet (repo + sim + dome metrics).
7. **Institution outreach** — curated list: labs, professors, foundations (name, title, email, focus).

## Brain parameter scaling roadmap

The pantheon's cognitive architecture scales on a defined ladder:

| Tier         | Scope                    | Parameters | Status   |
| ------------ | ------------------------ | ---------- | -------- |
| Glyph (×100) | Each letter creature     | 25,000     | **Live** |
| APEX-100K    | APEX #101 starting brain | 100,000    | **Live** |
| APEX-250K    | First expansion          | 250,000    | Designed |
| APEX-500K    | Second expansion         | 500,000    | Designed |
| APEX-1M      | Third expansion          | 1,000,000  | Designed |
| APEX-2.5M    | Fourth expansion         | 2,500,000  | Designed |
| APEX-5M      | Near-term roadmap target | 5,000,000  | Designed |
| MASSIVE      | Full brief target        | ~2.5B      | Roadmap  |

Total live brain compute: 100 × 25k (glyph) + 100k (APEX) = **2.6M parameters** running in-browser today.
The designed tiers (250k–5M) are architecture-ready; the live engine caps allocation at 4096 nodes/organ
for browser safety. Reaching the full designed scale requires a C/C++/GPU backend (roadmap).

## New ventures (horizon)

### V1 — Educational lab license

- Target: universities, high-school STEM, coding bootcamps
- Offer: hosted instance + curriculum packet (determinism, A-Life, active inference, quantum simulation)
- Price: $5/mo per .edu seat, free for verified non-profits
- Status: pricing sketched, curriculum not yet written

### V2 — Researcher API

- Target: A-Life, digital biology, NHSI researchers
- Offer: headless deterministic sim runs via CLI + JSON telemetry export
- Price: free for published research (attribution required), $50/mo for commercial use
- Status: `bun test` headless mode works today; API wrapper not built

### V3 — Art toy / collectible

- Target: generative art collectors, NFT-curious, indie game devs
- Offer: deterministic seed → unique creature + dome render → mintable SVG/PNG
- Price: $10 per seed (one-time), cosmetic drops for GitHub stars
- Status: SVG export works; minting pipeline not built

### V4 — Streamer overlay

- Target: Twitch/YouTube streamers in science, coding, gaming
- Offer: OBS browser source that shows live sim + chat-driven chaos/mutate controls
- Price: free with attribution, $5/mo for custom seeds + branding
- Status: not started; sim already renders in a canvas

### V5 — Peer-review meta-analysis

- Target: academic journals (ALife, Frontiers, PLOS Computational Biology)
- Offer: falsifiable comparison against ~100 A-Life projects using the NHSI 14/14 indicator framework
- Price: N/A (research output)
- Status: indicator framework documented; study not yet conducted
