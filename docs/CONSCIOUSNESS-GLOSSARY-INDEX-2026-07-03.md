# Consciousness Glossary Index

**Date:** 2026-07-03  
**Purpose:** term contract for Cosmogonic consciousness-lab reports, code comments, UI labels, and GitHub Pages copy.

## Claim Status Words

| Term                     | Allowed Meaning                                                                                                         |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------- |
| Computational indicator  | A measurable mechanism associated with a consciousness theory. It is not experience.                                    |
| Structural presence      | Code contains the named mechanism. It may still be weak or partial.                                                     |
| Causal wiring            | A mechanism changes behavior, learning, survival, attention, action, or another measured downstream state.              |
| Partial                  | A real mechanism exists, but it is thin, proxy-like, incomplete, or not yet strong enough.                              |
| Met                      | The mechanism is present, causal, tested, and ablation-sensitive.                                                       |
| Phenomenal consciousness | Subjective experience. Not claimed.                                                                                     |
| Sentience                | Capacity for felt valence/experience. Not claimed unless a future science standard exists.                              |
| Proto-sentience          | Project direction only: computational scaffolding that may be relevant to sentience research.                           |
| Proof                    | In this project, proof means deterministic code receipt plus test plus falsifier; it does not mean proof of experience. |
| Singularity moment       | A replayable multi-framework convergence event; interesting evidence, not a consciousness proof.                        |

## Architecture Terms

| ------------------------ | -------------------------------------------------------------------------------------------------- |

| Ignition                 | Thresholded workspace event where a coalition wins and broadcasts.                                 |
| Attention                | Allocation of limited processing resources.                                                        |
| Attention schema         | Internal model of attention, separate from attention itself.                                       |
| Self-model               | Predictive model of the entity's own state, limits, and control relation to the world.             |
| Metacognition            | Monitoring and control of internal cognition, confidence, and error.                               |
| Valence                  | Good/bad/need signal that biases learning and action.                                              |
| Allostasis               | Regulation by anticipating needs rather than merely reacting.                                      |
| Homeostasis              | Regulation around viable internal ranges.                                                          |
| Autopoiesis              | Self-producing, self-maintaining organization.                                                     |
| Embodiment               | Causal loop between action, body, and incoming sensation.                                          |
| Enaction                 | Cognition constituted through active world engagement.                                             |
| Sensorimotor contingency | Lawlike relation between action and sensory change.                                                |
| Affordance               | Action possibility offered by the world to the entity.                                             |
| UAL chain                | Compound, delayed, cumulative associative learning sequence.                                       |
| Temporal thickness       | Ability to associate states separated in time.                                                     |
| Projective frame         | Viewpoint-centered coordinate frame for field-of-consciousness modeling.                           |
| Field of consciousness   | PCM term for viewpoint-organized conscious space; in Cosmogonic, a modelled viewport, not a claim. |
| Phi proxy                | Tractable irreducibility/integration metric inspired by IIT.                                       |
| Partition                | Split of a system used to test whether whole-system causal power is lost.                          |
| Resonance                | Phase/frequency coordination among modules or entities.                                            |
| Coherence                | Degree of phase/state alignment; must be causal to matter.                                         |
| Criticality              | Near-transition regime where small changes can propagate widely.                                   |
| Metastability            | Stable-enough but flexible switching among coordinated states.                                     |
| Non-equilibrium          | Persistently maintained far-from-random, far-from-dead dynamics.                                   |
| Causal spine             | Graph of measured mechanism-to-behavior influence.                                                 |
| Ablation                 | Removal/disablement of a mechanism to test whether it matters.                                     |
| Null baseline            | Control agent or shuffled metric expected not to show the target effect.                           |
| Seed replay              | Deterministic rerun proving a trace is reproducible.                                               |

## Forbidden Shortcuts

- Do not write "is conscious" for a simulation trace.
- Do not write "proved sentience" for indicator convergence.
- Do not count a named module as wired unless its output changes downstream behavior or state.
- Do not count a source citation as implementation.
- Do not count a visualization as evidence unless it is generated from measured data.
- Do not hide weak or partial indicators behind aggregate scores.
- Do not let one metric dominate the verdict.
- Do not make LLM text fluency the proxy for consciousness.

## Preferred Sentence Patterns

- "This mechanism models X and is tested by Y."
- "This score is a computational indicator, not a sentience claim."
- "The effect is load-bearing because ablation reduces Z."
- "The current evidence supports structural presence, not full satisfaction."
- "This is a replayable convergence event worth studying, not proof of subjective experience."
- "The null baseline failed/passed; therefore the claim is strengthened/weakened."
