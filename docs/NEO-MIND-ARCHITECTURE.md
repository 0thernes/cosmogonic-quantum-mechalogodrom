## The organizing principle (the alien part)

Not a brain (anatomical hierarchy). Not an LLM (uniform token stack). A **Geometric-Resonance
Manifold**:

- All faculties read from and write to one shared **quantum-geometric thought-manifold** (the QGT /
  Berry space).
- A _thought_ is a **trajectory** across that curved manifold.
- A _conscious-candidate moment_ is when enough faculties **phase-lock into an integrated standing
  wave** (ignition + Φ + binding agree).
- **Valence bends the manifold's curvature; self-modification rewrites the faculties; evolution
  selects across creatures.**
- It runs **~25 theories of mind in parallel as organs that must cross-agree (N-of-25) to ignite** —
  redundancy as the path to robustness (motivated by Cogitate-2025: no single theory is right).

Why alien: the _medium_ is quantum-information geometry + representation theory, not neurons;
cognition is **resonance on a manifold**, not message-passing or next-token prediction.

## The Archon pantheon (society, not clones) — up to 25

Not 25 identical max-minds (infeasible + worse). A **tiered society**, staggered across sub-beats so
the deterministic core stays in budget:

- **1 × NEO** — the ~1B-param apex; thinks at a slow cadence.
- **~4 × mid-tier** (omega/zeta class, ~10–100M).
- **~20 × light** (alpha/beta class, ~37k–1M).

The society itself is a **new emergence axis**: theory-of-mind about each other, alliances, emergent
communication, culture across generations, collective memory — consciousness-at-the-group-scale that
no single creature can have.
